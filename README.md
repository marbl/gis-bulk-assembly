THIS REPO IS NOT PUBLICALLY SUPPORTED

Collection of scripts to download/process/assemble large number of samples via verkko. Typically used for HPRC releases.

See included tsv files in hprc-cache on how input data is described. See also https://github.com/skoren/R3_HPRC for updated examples and https://github.com/NHGRI/marbl_utils/blob/master/sequence_tools/convertToHPRCFormat.py for a script to generate a compatible tsv from R3 inputs.

See config.ini for how to set up information for assemblies/software/data location. Multiple instances can be run using the same data by setting up different versions.

See hprc.txt for info on the actual commands available and example usage.
